## 🔧 Para Desarrolladores

### Personalización de Colores
Edita las variables CSS en `styles.css` líneas 2-15:
```css
:root {
    --primary-color: #1a365d;    /* Azul principal */
    --accent-color: #f7941e;     /* Naranja/amarillo */
    /* ... más colores */
}
```

### Contenido
- **Textos**: Modifica directamente en `index.html`
- **Imágenes**: Reemplaza archivos en la carpeta `assets/`
- **Información de contacto**: Actualiza en las secciones correspondientes

### JavaScript
- **Configuración**: Ajusta parámetros en `script.js`
- **Parallax**: Modifica los valores `data-parallax` en el HTML
- **Formulario**: Conecta con tu backend editando la función `handleFormSubmission`
