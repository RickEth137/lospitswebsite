# Assets para LOS PITS Website

## Estructura de Carpetas

```
assets/
├── logo-los-pits.png           # Logo principal de la empresa
├── software-screenshot.png     # Captura de pantalla del software de gestión
└── logos/                      # Logos de marcas asociadas
    ├── toyota-logo.png
    ├── freightliner-logo.png
    ├── international-logo.png
    ├── vw-logo.png
    ├── jac-logo.png
    ├── yale-logo.png
    └── kenworth-logo.png
```

## Imágenes Necesarias

### 1. Logo Principal
- **Archivo**: `logo-los-pits.png`
- **Dimensiones**: 200x200px mínimo
- **Formato**: PNG con fondo transparente
- **Descripción**: Logo de la empresa LOS PITS

### 2. Screenshot del Software
- **Archivo**: `software-screenshot.png`
- **Dimensiones**: 800x600px mínimo
- **Formato**: PNG o JPG
- **Descripción**: Captura de pantalla del sistema de gestión de mantenimiento

### 3. Logos de Marcas
Todos los logos deben ser:
- **Dimensiones**: 150x80px aprox (manteniendo proporción)
- **Formato**: PNG con fondo transparente
- **Calidad**: Alta resolución para web

#### Marcas requeridas:
- Toyota
- Freightliner
- International
- Volkswagen
- JAC
- Yale
- Kenworth

## Imágenes de Fondo (Ya configuradas desde Unsplash)

El sitio web utiliza imágenes de fondo de alta calidad desde Unsplash:

1. **Hero Section**: Taller mecánico profesional
2. **About Section**: Textura metálica/maquinaria
3. **Services Section**: Motor de camión/piezas mecánicas
4. **Technology Section**: Fondo tecnológico abstracto

## Optimización de Imágenes

Para un mejor rendimiento, asegúrate de:

1. **Comprimir** todas las imágenes para web
2. **Usar WebP** cuando sea posible
3. **Mantener** el tamaño de archivo bajo 500KB por imagen
4. **Generar** versiones responsive si es necesario

## Herramientas Recomendadas

- **TinyPNG**: Para comprimir imágenes
- **Canva**: Para crear logos placeholder
- **Figma**: Para diseñar elementos gráficos
- **Remove.bg**: Para eliminar fondos de logos

## Notas Importantes

- Las imágenes de fondo actuales son temporales de Unsplash
- Los logos de marcas deben obtenerse de fuentes oficiales
- Asegúrate de tener los derechos de uso de todas las imágenes
- El logo principal debe reflejar la identidad visual de LOS PITS
